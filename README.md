# Loan_Approval_System
Loan Approval System or predictor is a machine learning mini project. It predicts weather to approve a loan on the basis of some given informations ( inputs ).
An end-to-end supervised ML pipeline using KNN, Logistic Regression, Naive Bayes to predict loan approval.\n Implemented Binary classification along with EDA, data scaling feature engineering & model evaluation (Precision, Recall, F1 score)
